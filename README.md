¿Qué es TermuxForge?
TermuxForge transforma tu teléfono Android en un entorno de desarrollo Linux completo. No es una aplicación, no requiere root, no flashlea nada. Es un script que instala y configura todo paso a paso, dejándote elegir exactamente qué necesitas.

¿Qué NO es?
No es una distribución Linux. No es necesario reemplazar Android.
No requiere root. Funciona en cualquier Android 7+ con Termux.
No instale nada sin preguntar. Cada herramienta requiere confirmación.
No es un guión monolítico. Cada módulo es independiente.
